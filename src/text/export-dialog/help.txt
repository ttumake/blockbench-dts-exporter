@@quick_start
Quick Start
1. Model and texture the asset in Blockbench.
2. Add helper nodes like muzzlePoint only if the target game or script needs them.
3. Use Mixed Materials for a blend of simple swatches and preserved source textures.

@@export_settings
Export Settings
- DTS Version: DTS 24 is the default and what Blockland uses.
- Orientation: Blockland Orientation applies the exporter’s game-facing axis remap. None leaves the model as-is.

@@materials
Materials
- Mixed Materials: keeps detailed textures where needed and converts simple source textures into generated swatches.
- Color Swatches: converts everything it can into generated swatches.
- Full Textures: preserves source texture assets and UV layout.
- Glow is controlled per material in the Materials tab.
- Global flags affect every exported material unless the format handles that property per material.

@@sequences
Sequences
- Blockbench animations are exported as DTS sequences.
- Position, rotation, and scale tracks are supported.
- Sequence markers are exported as triggers.
- Keep moving parts as separate objects or clearly separated animated groups.

@@lod_conventions
LODs
- Create groups named like detail128, detail64, detail32.
- Keep matching object names across LOD groups so variants pair correctly.
- If no detailNN groups are present, export falls back to a single default detail level.
